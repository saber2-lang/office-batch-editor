# 批量内容替换工具

批量替换 Word / Excel 文件内容、文件名、目录名的桌面工具，支持可视化界面与命令行两种操作方式。

支持的文件格式：`.doc` `.docx` `.xlsx` `.xls`

---

## 环境要求

- **操作系统**：Windows
- **Python**：3.8 及以上（推荐 3.10 / 3.11）

---

## 安装依赖

```bash
pip install customtkinter python-docx openpyxl "xlrd==1.2.0" xlwt xlutils pywin32
```

> **说明**
> - `xlrd` 必须使用 `1.2.0`，高版本不支持 `.xls` 格式
> - `pywin32` 仅用于处理 `.doc` 文件，且需要本机已安装 Microsoft Word；不处理 `.doc` 可跳过

---

## 文件说明

| 文件 | 说明 |
|------|------|
| `gui.py` | 可视化界面入口（推荐） |
| `rename_tool.py` | 核心逻辑，同时支持命令行运行 |

---

## 启动方式

### 可视化界面（推荐）

```bash
python gui.py
```

界面启动后：

1. **选择目标目录**：点击「浏览…」选择需要处理的文件夹；若只处理单个文件，勾选「仅处理单个文件」后选择文件
2. **填写替换规则**：在「查找内容」栏输入要替换的旧文字，在「替换为」栏输入新文字（留空表示删除）；点击「＋ 添加规则」可添加多条规则
3. **选择替换模式**：
   - 文件内容：只修改文件内部文字
   - 文件名/目录名：只重命名文件和目录
   - 两者都替换：同时执行
4. **点击「🔍 预览」**：先预览将被修改的文件，确认无误后点击「✅ 执行替换」

### 命令行

```bash
# 交互模式
python rename_tool.py

# 非交互模式（指定目录、规则、直接执行）
python rename_tool.py --dir "C:\体系文件" --old "旧公司名" --new "新公司名" --mode 1 --yes
```

---

## 注意事项

- 处理 `.doc` 文件前请先关闭 Word，避免文件冲突
- 建议勾选「执行前创建备份」，首次使用时尤为重要
- 工具会自动跳过 Office 临时文件（`~$` 开头）、空文件和备份目录
